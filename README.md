# Cyber Security Labs & Projects

This repository contains my hands-on cybersecurity practice labs, tools, and notes related to networking and security monitoring.

## Skills Practiced

* Network Scanning
* Packet Analysis
* Security Monitoring
* Vulnerability Assessment
* Reconnaissance Techniques

## Tools Used

* Kali Linux
* Nmap
* Wireshark
* Wazuh SIEM
* pfSense Firewall
* VirtualBox

## Lab Environment

A virtual cybersecurity lab was created using VirtualBox where multiple machines were monitored using Wazuh agents.
The network traffic was controlled using pfSense firewall and security logs were collected for monitoring and analysis.

## Repository Contents

### Information Gathering Tool

A simple reconnaissance tool that collects domain and network intelligence useful for identifying attack surfaces.

### Cybersecurity Monitoring Lab

A simulated environment where security logs are collected and analyzed using Wazuh.

### Networking & Security Notes

Notes related to networking fundamentals, ports, protocols, and security concepts.
